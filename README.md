# E-Portfolio — Grâce Mutata

Ce projet est un portfolio personnel développé pour présenter mon parcours,
mes compétences, mes réalisations et permettre aux clients de me contacter.
Il est entièrement réalisé en HTML et CSS et est optimisé pour les écrans
mobile, tablette et desktop.

---

## Fonctionnalités

- Page d'accueil présentant mon profil
- Page CV avec mon parcours scolaire et professionnel
- Page Projets avec mes réalisations
- Page Contact avec formulaire
- Design responsive (mobile/tablette/desktop)

---

## Technologies utilisées

- **HTML5** — Structure sémantique du site  
- **CSS3** — Mise en page, design, responsive  
- **Git & GitHub** — Versioning et hébergement  
- **GitHub Pages** — Mise en ligne du portfolio
